## About

This portfolio showcases the websites, web applications, and tools I have built using modern technologies. It serves as a place for recruiters, clients, and fellow developers to explore my work, technical skills, and development experience.

## Features

- Featured Projects: Showcase of my work with project details and links.
- Technical Skills: Overview of the technologies and tools I use.
- Responsive Design: Optimized for desktop, tablet, and mobile devices.
- Modern UI: Clean layouts, smooth interactions, and accessible design.
- Dark Mode: Support for both light and dark themes.

## Tech Stack

**Core Architecture**

- Next.js (App Router)
- React
- TypeScript

**Styling & UI**

- Tailwind CSS 4
- Framer Motion
- Radix UI Primitives
- Lucide React & React Icons
- clsx & tailwind-merge

## Purpose

This project serves as my professional portfolio, showcasing the websites, web applications, and tools I have built. It reflects my approach to creating fast, scalable, and user-friendly software using modern web technologies.
